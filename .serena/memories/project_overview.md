# ORPulse Landing Page V2

## Project Purpose
A modern, responsive landing page for the ORPulse productivity ecosystem that showcases the company's suite of applications.

## Tech Stack
- **Framework**: Vue 3 with Composition API (`<script setup>` syntax)
- **Language**: TypeScript
- **Build Tool**: Vite
- **CSS Framework**: Tailwind CSS v4
- **Icons**: Lucide Vue Next
- **Deployment**: Docker with nginx

## Key Features
- Dark/Light mode toggle with localStorage persistence
- Responsive mobile-first design
- Single Page Application (SPA) architecture
- SEO optimized with meta tags
- Smooth scroll navigation to sections

## Applications Showcased
1. **Chronor** (Live): Time management app at chronor.orpulse.app
2. **TimeLogger** (In Development): Professional time tracking
3. **Project Hub** (Coming Soon): Project management platform
