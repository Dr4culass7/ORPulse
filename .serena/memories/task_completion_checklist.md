# Task Completion Checklist

When completing any development task:

1. **Type Checking**: Run `npm run build` to ensure TypeScript compilation passes
2. **Visual Testing**: Run `npm run dev` and test in browser
3. **Responsive Design**: Check mobile, tablet, and desktop views
4. **Dark Mode**: Verify both light and dark themes work correctly
5. **Browser Testing**: Test in Chrome/Firefox/Safari if possible
6. **Git Commit**: Create meaningful commit messages following the pattern from git log

## Quality Checks
- No TypeScript errors
- Tailwind classes are valid
- Components follow existing patterns
- Dark mode variants included where needed
- Accessibility considerations (aria-labels, semantic HTML)
